# Linux Basic Commands

#### Folders

Show content of a folder

```text
ls
ll
ls -all
```

Change to another folder

```text
cd
cd /var/www/
cd .. 
```

Make a new Folder

```text
mkdir
```

Remove a folder

```text
rmdir
```

Remove all files and folders in a folder

```text
rm -rf *
```

Show your current location

```text
pwd
```

Size of a folder

```text
du
```

#### Files

Make a new file

```text
vim newfile.txt
```

Remove a file

```text
rm newfile.txt
```

Move or rename a file

```text
mv 
```
